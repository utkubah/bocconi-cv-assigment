# Bocconi IPCV — Two-View Geometry Assignment

This repo contains our notebook for the IPCV homework on two-view geometry:
camera calibration, feature matching, essential matrix estimation, epipolar lines, and 3D reconstruction.

## Contents
- `computer_vision.ipynb` — main notebook (run this)
- `data/` — input images
  - `image1.jpeg`, `image2.jpeg` (Images of Knowledge That Matter sign from 2 different angles)
  - `calibration/` (checkerboard images for calibration)
- `calibration_results.npz` — saved calibration output (generated by the notebook)
- `reconstruction_E_ours.npz` — saved reconstruction output (generated by the notebook)
- `requirements.txt` — Python dependencies
